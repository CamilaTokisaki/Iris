## Installation

Well, that's easy:   
Install [Python](https://www.python.org/)  
Download the archive, unpack it, open the unpacked folder in a terminal and write 
```py
pip install -r requirements.txt
```
After successful installation, start the project 
```py
python .\Iris.py
```
