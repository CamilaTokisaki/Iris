## Установка

Ну тут всё просто:   
Устанавливаем [Python](https://www.python.org/)  
Скачиваем архив, распаковываем, открываем в терминале расспакованную папку и пишем 
```py
pip install -r requirements.txt
```
После удачной установки запускаем проект 
```py
python .\Iris.py
```
